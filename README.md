# Portrait Photography — Vintage Temple Theme

A Next.js portrait photography site with a vintage temple-inspired layout: warm sepia palette, elegant serif headings, soft sans body text, aged-paper background, smooth scroll, and a subtle grain overlay.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Design

- **Colors:** Sepia, deep brown, muted gold, stone gray (CSS variables in `app/globals.css`)
- **Typography:** Cormorant Garamond (headings), Source Sans 3 (body) via `next/font`
- **Background:** Aged paper / stone texture; optional grain overlay
- **Behavior:** Smooth scroll; reduced motion respected
- **SEO:** Metadata and Open Graph in `app/layout.tsx`

## Build

```bash
npm run build
npm start
```
